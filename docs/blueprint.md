# **App Name**: HostInsight Dashboard

## Core Features:

- Server Overview Display: Present a concise summary of all managed servers (Dell R740, R740XD, R720), their current status, and key identifying information such as IP address and model.
- Detailed Server Pages: Provide dedicated, interactive pages for each server, detailing their hardware specifications (CPU, RAM, Storage), EXSi version, and network configuration.
- Virtual Machine Listings: For each server, display a clear, tabular list of hosted virtual machines, including VM status, used storage space, guest OS, host name, CPU, and allocated memory, similar to the provided spreadsheet image.
- Interactive Server Navigation: Implement three distinct buttons allowing users to easily navigate and switch between the detailed pages of the different Dell servers.
- Dynamic Status Indicators: Visually represent VM and server status using color-coded indicators (e.g., red/green/yellow as seen in the source image) for quick recognition of operational state.
- Intelligent Server Health Assistant: A tool that analyzes current server and virtual machine metrics to provide proactive recommendations, flag potential performance bottlenecks, or suggest optimization strategies.

## Style Guidelines:

- Dark scheme background: A very dark desaturated blue (#14141F) to provide a modern, high-tech dashboard feel, optimizing for long viewing sessions and reduced eye strain.
- Primary interactive color: A rich, deep blue-purple (#2626D9) for buttons, active elements, and data highlights, conveying reliability and professionalism.
- Accent and call-to-action color: A bright, clear cyan-blue (#66C6F5) used sparingly for critical alerts, primary call-to-action buttons, or to draw attention to important metrics.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clean appearance and excellent readability across various data types and densities typical of a dashboard application.
- Utilize minimalist, system-style icons to represent server components, virtual machines, and network elements, ensuring clear visual communication without clutter.
- Responsive, grid-based layout for data tables and component cards to ensure clarity and optimal information display on various screen sizes, from desktop to mobile.
- Subtle and quick transitions between server pages and on data updates to enhance responsiveness and user feedback without being distracting.
